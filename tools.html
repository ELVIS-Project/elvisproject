Interval Sonifier

Exploring databases of music lends itself naturally to an auditory representation. Our interval sonifier allows you to listen to your data, creating unexpected insights and musical meaning. Available as a standalone application to users of MacOSX 10.6-10.8, it can also run on any platform running SuperCollider.

music21

Music21 is a python-based library for computer-aided musicology. An invaluable collaboration in the ELVIS project, it has a developed network of users and substantial documentation. Music researchers can benefit from the host of tools available for computational analysis.
